Comment Lancer le Backend (Django API)Le Backend s'exécute sur http://127.0.0.1:8000.ÉtapeCommande (dans le dossier racine du projet)RôleA. Environnement Virtuelpython -m venv venvCrée l'environnement isolé.B. Activationsource venv/bin/activate (Mac/Linux) ou .\venv\Scripts\activate (Windows)Isole les dépendances Python.C. Installer Dépendancespip install -r requirements.txtInstalle DRF, Cloudinary, etc.D. Appliquer Migrationspython manage.py migrateCrée la base de données.E. Lancement du Serveurpython manage.py runserverDémarre l'API sur http://127.0.0.1:8000.3. Comment Lancer le Frontend (React App)Le Frontend s'exécute sur http://localhost:3000.ÉtapeCommande (dans le dossier frontend/)RôleA. Installationnpm installInstalle les packages React, tawk-messenger-react, etc.B. Lancementnpm startDémarre l'application en mode développement.4. Comment Créer un SuperutilisateurPour vous donner un accès au panneau d'administration de Django :ÉtapeCommande (dans le dossier backend/)RésultatCréationpython manage.py createsuperuserCrée un compte avec tous les privilèges (Admin).5. Comment Tester les APIsL'API utilise l'authentification JWT. Les requêtes doivent être testées avec un outil comme Postman ou cURL.Obtenir le Token (Connexion) :Méthode : POSTURL : http://127.0.0.1:8000/api/token/Body (JSON) : {"username": "votre_nom", "password": "votre_mot_de_passe"}Résultat : Vous recevez un access token.Tester un Endpoint Protégé (Liste des Tickets) :Méthode : GETURL : http://127.0.0.1:8000/api/tickets/Headers : Ajoutez le header Authorization: Bearer VOTRE_ACCESS_TOKEN.Résultat : L'API retourne la liste des tickets (si vous êtes Admin, vous voyez tout ; si vous êtes Client, vous voyez vos propres tickets).6. Comment Tester l'Intégration du Chat (Tawk.to)L'intégration doit être vérifiée côté Frontend et côté Tableau de bord Tawk.to.Vérification Frontend :Ouvrez le navigateur à l'adresse http://localhost:3000.L'icône de chat Tawk.to doit être visible en bas à droite, sur toutes les pages.Vérification de la Session :Cliquez sur le widget et envoyez un message de test (ex: "Bonjour, j'ai une question").Connectez-vous à votre tableau de bord Tawk.to et vérifiez que vous recevez la conversation en temps réel.7.
